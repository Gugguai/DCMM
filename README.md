# 以海尔公司为例进行数据治理分析 (DCMM Case Study: Haier)

本项目是一个基于 Streamlit 的交互式 Web 应用，旨在通过 DCMM (数据管理能力成熟度评估模型) 视角，深入分析海尔集团在数据治理方面的发展历程、组织架构变革及制度建设成就。

## 📖 项目背景

本项目为小组课程作业，通过收集海尔集团的公开资料（年报、新闻报道、官方网站等），梳理了海尔从 2012 年至今的数字化转型与数据治理之路，重点分析了其从“受管理级”到“优化级”的演进过程。

## 🚀 功能模块

应用包含以下主要板块：

1.  **作业流程**：展示项目从选题到报告撰写的全过程。
2.  **任务分工**：明确小组成员在数据收集、可视化、开发等环节的职责。
3.  **公司介绍**：海尔集团概况、组织架构（倒三角）、股权结构及核心业务（智慧住居、产业互联网、大健康）。
4.  **数据治理分析**：核心板块，按时间轴深度解析：
    *   **2012年以前（初始级）**：分散管理，缺乏统一标准。
    *   **2012-2014（受管理级）**：网络化战略，互联工厂探索，U+平台发布。
    *   **2015-2018（稳健级）**：DTS部门成立，COSMOPlat平台上线，灯塔工厂布局。
    *   **2019-2021（量化管理级）**：数据管理委员会，海数汇平台，生态覆盖。
    *   **2022至今（优化级）**：持续优化与成就。
5.  **总结**：项目总结与展望。

## 🛠️ 技术栈

*   **Python 3.8+**
*   **Streamlit**: 用于构建交互式 Web 界面。

## 💻 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/Gugguai/DCMM.git
cd DCMM
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行应用

```bash
streamlit run app.py
```

应用将在浏览器中自动打开，默认地址为 `http://localhost:8501`。
